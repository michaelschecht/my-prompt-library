# Git Rebase Skill

## Prerequisites
- Git installed

## Usage
`git rebase -i HEAD~3`

## Configuration
None

## Examples
Rebasing 3 commits to squash them.

## Advanced Usage
`git rebase --onto master feature bugfix`

## Integration
- IDE Git integrations

## Troubleshooting
- Merge conflicts? Resolve and `git rebase --continue`.

## Best Practices
- Don't rebase public history.
