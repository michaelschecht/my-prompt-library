# Markdown Formatter

Formats messy markdown.

## Prerequisites
- Prettier

### Setup Instructions
1. npm install -g prettier

**Verification:**
```bash
prettier --version
```

Expected output:
```
3.0.0
```

## Usage
### Basic Usage
```bash
prettier --write file.md
```
**What it does:**
1. Formats file.

## Configuration
### Optional Configuration
None.

## Examples
### Example 1
**Context:** Messy file.
**Task:** Format it.
**Commands:**
```bash
prettier --write .
```
**Output:**
```
Formatted
```
