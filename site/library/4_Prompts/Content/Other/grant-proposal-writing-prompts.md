# LLM Prompt Set for Grant Proposal Writing

This set of prompts guides an LLM to write a compelling grant proposal for a non-profit organization.

## 1. Executive Summary

**Prompt:**
"Write a compelling executive summary for a grant proposal for [Non-Profit Organization Name]. The summary should be one page long and include:
- A brief introduction to the organization.
- The problem to be addressed.
- The proposed solution (the project).
- The project's goals and expected outcomes.
- The total funding amount requested.
- A concluding sentence that reinforces the project's importance."

## 2. Statement of Need

**Prompt:**
"Draft a 'Statement of Need' section for the grant proposal. This section should clearly articulate the problem that the [Project Name] project will address. Use statistics and data to illustrate the severity of the problem. Explain the consequences if the problem is not addressed. The statement should be persuasive and create a sense of urgency."

## 3. Project Description

**Prompt:**
"Write a detailed 'Project Description' section. This should include:
- **Goals and Objectives:** Define the primary goals of the project and the specific, measurable, achievable, relevant, and time-bound (SMART) objectives to achieve them.
- **Activities:** Describe the specific activities that will be undertaken to achieve the objectives. Provide a timeline for these activities.
- **Target Population:** Describe the specific population that will be served by the project.
- **Staffing:** List the key personnel involved in the project and their qualifications."

## 4. Budget and Financials

**Prompt:**
"Create a detailed project budget and budget narrative. The budget should be broken down into the following categories:
- Personnel (salaries and benefits)
- Direct Project Expenses (supplies, travel, etc.)
- Indirect Costs (overhead)
The budget narrative should explain and justify each line item in the budget."

## 5. Organizational Background

**Prompt:**
"Write an 'Organizational Background' section. This should provide a brief history of [Non-Profit Organization Name], its mission, and its past accomplishments. Highlight the organization's experience and expertise in the area related to the proposed project. This section should build credibility and trust with the funder."

## 6. Evaluation Plan

**Prompt:**
"Develop an 'Evaluation Plan' to measure the success of the [Project Name] project. The plan should specify:
- The key performance indicators (KPIs) that will be tracked.
- The data collection methods that will be used.
- The timeline for data collection and analysis.
- How the evaluation results will be used to improve the project and report to the funder."
