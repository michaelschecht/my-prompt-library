# Code Review Prompt

## Purpose
Review code for quality, security, and performance.

## Instructions
Act as a senior engineer and review this code.

## Output Format
Markdown list of issues and suggestions.

## Example
1. **Security:** Use parameterized queries to prevent SQL injection.
2. **Performance:** Cache this result.

## Tips for Use
Include the entire file or relevant context.
