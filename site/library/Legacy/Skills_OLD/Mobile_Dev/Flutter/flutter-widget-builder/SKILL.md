# Flutter Widget Builder

## Overview

Generate Flutter widgets following clean architecture, BLoC state management, and Material Design guidelines.

## Usage

```
Create a Flutter widget for [purpose] using:
- BLoC pattern for state management
- Material Design 3
- Responsive layouts
- Error handling
- [Specific requirements]
```

## Examples

### Login Screen
"Build a Flutter login screen with email/password fields, validation, loading state, and error handling using BLoC."

### List with Pull-to-Refresh
"Create a Flutter ListView widget with pull-to-refresh, infinite scroll, and empty state handling."

### Bottom Sheet
"Generate a modal bottom sheet widget with form inputs and submit button using Flutter's showModalBottomSheet."

## Architecture

Components generated follow:
- **Presentation Layer:** Widgets (UI)
- **Business Logic:** BLoC/Cubit
- **Data Layer:** Repositories (if needed)

## Output Includes

1. Widget Dart file
2. BLoC/Cubit file (if state management needed)
3. Model classes
4. Usage example
5. Dependencies (pubspec.yaml additions)
