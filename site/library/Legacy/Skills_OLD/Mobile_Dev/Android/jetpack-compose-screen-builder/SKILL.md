# Jetpack Compose Screen Builder

## Overview

Generate Android screens with Jetpack Compose following modern Android architecture (MVVM), Material Design 3, and Kotlin coroutines.

## Usage

```
Create an Android Compose screen for [purpose]:
- Jetpack Compose UI
- ViewModel with StateFlow
- Hilt dependency injection
- Material 3 design
- [Specific features]
```

## Examples

### User Profile
"Build a Compose profile screen with user info, stats, and action buttons. Use ViewModel with StateFlow for state management."

### Form Input
"Create a Compose form screen with text fields, dropdowns, validation, and submit button. Include loading and error states."

### Bottom Navigation
"Generate a Compose bottom navigation setup with 3 tabs, each with its own NavHost."

## Architecture Components

- **Composables:** UI functions
- **ViewModel:** State holder with StateFlow/LiveData
- **Repository:** Data layer (if needed)
- **Hilt:** Dependency injection

## Output Includes

1. Composable function(s)
2. ViewModel with state
3. UI state sealed class
4. Preview composables
5. Hilt annotations
6. Navigation setup (if multi-screen)
