# Software License Agreement Template

## Purpose
Draft a software license agreement tailored to commercial model, permitted use, IP protections, and support terms.

## Instructions
Act as software licensing counsel.

Create a license agreement using:
- **Licensor/Licensee:** [entities]
- **License type:** [perpetual/term/subscription]
- **Scope:** [users/devices/territory]
- **Deployment:** [cloud/on-prem]
- **Restrictions:** [reverse engineering, redistribution, etc.]
- **Support/SLA:** [details]
- **Fees and payment terms:** [details]

Cover grant, restrictions, ownership, compliance audits, warranties, indemnity, limitation of liability, and termination effects.

## Output Format
Provide a clause-by-clause draft plus a negotiation notes table identifying high-risk clauses and fallback positions.
