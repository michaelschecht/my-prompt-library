## 8. Content Quality Guidelines

For all new additions, ensure:

### ✅ Required Elements
- **Clear title with emoji** (matching existing style)
- **Comprehensive tags** (3-7 relevant tags)
- **Proper categorization** (section, category, subcategory)
- **Detailed description** (what it does, when to use it)
- **Practical examples** (real-world use cases)
- **Expected outputs** (what users should get)

### ✅ Best Practices
- **Actionable prompts** - Users should be able to copy and immediately use
- **Context-aware** - Provide background information when needed
- **Scalable** - Work for beginners and experts
- **Ethical** - Include disclaimers for sensitive domains
- **Maintained** - Update as AI capabilities evolve

### ✅ Avoid
- Overly generic prompts that don't add value
- Duplicate content (check existing library first)
- Outdated techniques no longer relevant
- Prompts that could be harmful or unethical
