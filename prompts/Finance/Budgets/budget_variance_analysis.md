# Budget Variance Analysis

Quickly calculates and explains variances between budgeted and actual figures.

---

## Prompt

Analyze the following budget vs. actual data for [DEPARTMENT/PROJECT] over [TIMEFRAME].\n\nData:\n[BUDGET VS ACTUAL DATA]\n\n1. Calculate the absolute variance ($) and percentage variance (%) for each line item.\n2. Identify the top 3 largest unfavorable variances and the top 2 largest favorable variances.\n3. Provide a brief hypothesis or common business reasons for these specific variances.\n\nFormat the results in a clear markdown table.

---

## Output Structure

1. Summary Table (Item, Budget, Actual, Variance $, Variance %)\n2. Key Unfavorable Variances (Top 3)\n3. Key Favorable Variances (Top 2)\n4. Potential Causes / Recommendations
