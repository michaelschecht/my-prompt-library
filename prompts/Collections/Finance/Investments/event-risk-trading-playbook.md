# Event Risk Trading Playbook

## Objective
Create an event-driven trading plan for **[ASSET]** around **[EVENT]**.

## Inputs
- Asset: [ASSET]
- Event: [CPI / FOMC / EARNINGS / JOBS / OTHER]
- Time horizon: [PRE / POST / BOTH]
- Risk budget: [MAX % LOSS]

## Analysis Instructions
Include:
- Consensus expectations and surprise risk
- Historical reaction patterns for similar events
- Bull/base/bear scenarios with estimated probabilities
- Volatility expansion/compression expectations
- Best pre-event vs post-event approach

## Required Output
1. Scenario table with expected price ranges
2. Preferred setup before event
3. Preferred setup after confirmation
4. Strict no-trade conditions
5. Risk controls and position sizing guidance
