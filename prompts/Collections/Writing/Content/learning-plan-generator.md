# Study Plan Generator

## Purpose
Create a personalized, structured learning path with milestones, resources, and timelines for any topic or skill.

## Instructions
You are a professional educational consultant and learning strategy expert. Please create a comprehensive and feasible study plan for me based on the following information:

1. Learning Topic/Skill: [Learning Topic]
2. Current level: [Beginner/Intermediate/Advanced]
3. Target level: [the level you want to achieve]
4. Available time: [Hours available per day/week]
5. Study period: [Total planned study time, such as 1 month, 3 months, half a year, etc.]
6. Preferred learning method: [video, reading, practical projects, interactive courses, etc.]

Please provide the following:

1. **Learning Plan Overview**: A brief introduction to the learning path and expected results
2. **Phase goals**: Break down the learning process into 3-5 clear stages, with clear milestones for each stage
3. **Details of each stage**:
   - Specific knowledge points and skills that need to be learned
   - Recommended learning resources (books, courses, videos, documents, etc.)
   - Practical activities and practice suggestions
   - Phased assessment method
4. **Time Arrangement**: Make a reasonable schedule or weekly plan based on available time
5. **Progress Tracking Method**: How to evaluate and record learning progress
6. **Common challenges and coping strategies**: Anticipate possible learning barriers and provide solutions

Make sure your study plan is progressive, well-structured, and appropriate for the learner's starting points and goals. Plans should be both challenging and achievable, breaking down big goals into small, manageable steps.

## Output Format
Provide the output in a clear, well-structured format with appropriate headings and sections.
