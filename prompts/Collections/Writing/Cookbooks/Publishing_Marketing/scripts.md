# Phase 6: Publishing & Automation Scripts
**Goal:** Use Python/Pandoc to convert your library into formats.

**Prompt:**
"Write a Python script that takes a folder of Markdown files (recipes) and uses the Pandoc library to:
1. Batch convert them into a single, formatted PDF with a table of contents.
2. Generate an EPUB version optimized for Kindle.
3. Ensure all images referenced in the MD files are scaled to 300 DPI for print quality."
