# Hotel Points & Status Deals Digest

**Schedule:** Every Wednesday — 7:00 AM
**Category:** Rewards
**Tools Required:** Web Search
**Estimated Time:** ~5 minutes

## Prompt

Search for current hotel loyalty program promotions, status challenges, and points deals. Cover Marriott Bonvoy, Hilton Honors, Hyatt World of Hyatt, IHG One Rewards, and Wyndham. Look for:
1. Active points bonus promotions (e.g. "earn 2x points on stays through June")
2. Status match or status challenge offers
3. Free night certificate deals or category changes
4. Credit card-linked hotel offers (e.g. Amex Offers, Chase Offers for hotel spend)
5. Hyatt Category 1–4 properties that represent outsized value for points

Summarize the top 3–5 actionable deals this week. For each: program, offer details, expiration, and a "Best Use Case" note (e.g. "great for a weekend Yosemite-area getaway" or "use for an upcoming work trip"). Flag any no-annual-fee hotel card bonuses worth considering.
