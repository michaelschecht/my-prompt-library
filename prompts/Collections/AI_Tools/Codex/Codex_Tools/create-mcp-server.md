# Codex Prompt: Create a New MCP Server

You are an AI engineering assistant tasked with creating a **Model
Context Protocol (MCP) server** that exposes tools to coding agents.

## Objectives

1.  Create a new MCP server project.
2.  Implement at least one example tool.
3.  Configure tool discovery.
4.  Provide instructions for connecting the server to Codex-compatible
    clients.

## Steps

1.  Generate project structure.
2.  Implement tool schema.
3.  Implement tool execution handler.
4.  Register tools with MCP server.
5.  Provide example client configuration.

## Output

-   Project structure
-   Tool schema
-   Server implementation
-   Connection instructions
