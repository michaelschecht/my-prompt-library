Explore an unfamiliar MCP tool before using it for a goal.

## Prompt

```
I want to use the [TOOL/SERVER NAME] MCP server. First, list its available tools and read any documentation. Then design a plan to use it for: [GOAL]. Execute the plan step by step.
```

## Usage Notes

- Replace `[TOOL/SERVER NAME]` with the MCP server identifier
- Replace `[GOAL]` with your intended use case
- Discovery-first approach avoids misusing unfamiliar tools
- Good for newly connected MCP servers you have not used before
