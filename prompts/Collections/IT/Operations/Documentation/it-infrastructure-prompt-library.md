# 4. IT Infrastructure Prompt Library

This document contains categorized prompts for building IT Infrastructure documentation, standards, and operating procedures.

1. Create a server provisioning standard including hardware specifications, OS installation, and initial configuration steps.
2. Document a network topology diagram and accompanying description for data centers and cloud VPCs.
3. Develop a storage area network (SAN) or NAS configuration standard covering LUN provisioning, RAID levels, and zoning.
4. Write a standard operating procedure (SOP) for patch management across Windows and Linux servers.
5. Define a backup and restore policy covering frequency, retention, and testing procedures for all critical systems.
6. Create a physical data center access control procedure, including biometrics, visitor logs, and escort policies.
7. Document a standard for virtual machine (VM) lifecycle management: provisioning, decommissioning, and resource optimization.
8. Develop a standard for DNS and DHCP configuration, including naming conventions and redundancy.
9. Write a standard for network switch and router configuration, including VLANs, STP, and SNMP settings.
10. Define a capacity planning process for compute, storage, and network resources with forecasting models.
11. Create an SOP for hardware asset lifecycle: procurement, deployment, maintenance, and disposal.
12. Document a standard for remote access VPN configuration, including authentication methods and split tunneling policies.
13. Develop a standard for certificate management (PKI) including issuance, renewal, and revocation processes.
14. Write a standard for time synchronization (NTP) across all infrastructure devices.
15. Create a data center environmental monitoring standard (temperature, humidity, power) with alerting thresholds.
16. Document a standard for infrastructure monitoring tooling (e.g., Nagios, Zabbix, Prometheus) including agent configuration and alert routing.
17. Define a standard for infrastructure disaster recovery testing, including scope, frequency, and success criteria.
18. Develop a standard for physical server cabling and labeling in data centers.
19. Write a standard for email gateway configuration (anti-spam, anti-malware, TLS enforcement).
20. Create a standard for infrastructure as code (IaC) version control and CI/CD pipeline integration for infrastructure changes.
