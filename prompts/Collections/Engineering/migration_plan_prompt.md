# System Migration Plan (Based on AWS Migration Framework)

**Prompt:**
"Produce a migration plan for [System] using the AWS Migration Framework.
Include:
1. 6 Rs assessment (rehost, replatform, refactor, etc.).
2. Dependency mapping and critical path.
3. Cutover strategy and rollback plan.
4. Data migration steps and validation.
5. Risk register and success metrics.

Deliver a timeline with phases and checkpoints."
