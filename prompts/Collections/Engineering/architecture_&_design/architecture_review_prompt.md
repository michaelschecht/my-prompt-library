# Architecture Review (Based on SEI ATAM and IEEE 42010)

**Prompt:**
"Conduct an architecture review for [System] using SEI’s ATAM and IEEE 42010 concepts.
Include:
1. Stakeholders and quality attribute scenarios.
2. Architectural views (logical, process, deployment, data).
3. Sensitivity and tradeoff analysis.
4. Identified risks, non-risks, and mitigations.
5. Recommendations prioritized by impact and effort.

Deliver a structured report with diagrams described textually."
