# Prompt: Brainstorm New Feature Ideas

## Description
This prompt provides a template for brainstorming new feature ideas for a product. It encourages creative thinking by using the SCAMPER method.

## Template

### 1. Product
*   **Product Name:** My Awesome To-Do List App
*   **Target Audience:** Busy professionals who need to manage their tasks effectively.

### 2. SCAMPER

*   **S (Substitute):** What if we substitute the traditional to-do list with a Kanban board?
*   **C (Combine):** What if we combine the to-do list with a calendar to schedule tasks?
*   **A (Adapt):** What if we adapt the to-do list for different use cases, such as project management or event planning?
*   **M (Modify):** What if we modify the to-do list to include subtasks, deadlines, and priorities?
*   **P (Put to another use):** What if we use the to-do list to track habits or goals?
*   **E (Eliminate):** What if we eliminate the need for a separate notes app by adding a notes section to each task?
*   **R (Reverse):** What if we reverse the to-do list and show completed tasks first?

### 3. Feature Ideas

Based on the SCAMPER exercise, here are some potential feature ideas:

*   Kanban board view.
*   Calendar integration.
*   Project management templates.
*   Event planning templates.
*   Subtasks, deadlines, and priorities.
*   Habit tracking.
*   Goal tracking.
*   Notes section for each task.
*   "Reverse" to-do list view.
