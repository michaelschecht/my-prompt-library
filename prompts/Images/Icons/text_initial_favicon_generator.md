# Text-Initial Favicon Generator

Design goals:
- Modern, clean, minimalist
- Excellent legibility at 16x16 and 32x32
- Bold, geometric letterform with balanced spacing
- Avoid decorative fonts and fine details

Deliver standard favicon assets:
- `favicon.ico`
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png` (180x180)
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`
- `site.webmanifest`
- Optional: `favicon.svg`

Requirements:
- Provide light and dark theme-safe versions if needed
- Include exact hex palette and typography rationale
- Ensure crisp edges and centered composition

Output:
1. Concept A and B descriptions
2. Export file checklist with dimensions
3. Recommended winner with reasoning

5. QA checklist for browser tabs + mobile homescreen
