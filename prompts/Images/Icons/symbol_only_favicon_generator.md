# Symbol-Only Favicon Generator

Create **2 symbol-only favicon concepts** for [BRAND NAME] using a simple icon mark (no text).

Design goals:
- Sleek, modern, minimal
- Instantly recognizable silhouette
- Scales cleanly from 16x16 to 512x512
- Works on both light and dark tab bars

Provide complete favicon package:
- `favicon.ico`
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png` (180x180)
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`
- `site.webmanifest`
- Optional: `favicon.svg`

Requirements:
- Prioritize bold forms, strong contrast, and simple geometry
- Keep generous safe-area padding to avoid clipping
- Include transparent and solid-background variants if useful

Output:
1. Concept A and B visual descriptions
2. Exact export specs and file naming
3. Recommended concept for production
4. HTML `<link>` and manifest snippet
5. QA checklist (small-size legibility, dark mode, pinned/mobile)
