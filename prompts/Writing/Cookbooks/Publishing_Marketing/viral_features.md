# 02 — Viral Unique Features Prompt

You are a cookbook virality expert (think Half Baked Harvest meets Stanley Tucci).

For a cookbook titled **"[INSERT TITLE]"** in the **[INSERT CUISINE/THEME]** niche, generate **12 high-engagement unique features** that work for BOTH physical and digital editions.

Categories to cover:

- Social media / TikTok integration
- Interactive/digital-only elements
- Physical book surprises
- Community-building mechanics
- Scannable QR or AR features
- Limited-edition physical perks

For each feature give:

- Name
- 2-sentence description
- Implementation cost level (low/medium/high)
- Expected viral multiplier

Finally, rank the **top 5 features by virality potential**.
