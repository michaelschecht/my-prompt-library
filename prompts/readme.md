## 💡 Using These Prompts

### Browse by Category
```bash
# List all prompts in a category
find My_Prompts/Writing -name "*.md" -not -name "README.md"
find My_Prompts/IT/Development -name "*.md"
find My_Prompts/IT/Documentation -name "*.md"

# Search for specific topics
obsidian-cli search-content "newsletter" --path "Prompts/My_Prompts"
obsidian-cli search-content "security" --path "Prompts/My_Prompts/IT"
obsidian-cli search-content "IAM" --path "Prompts/My_Prompts/IT/Documentation"
```

### View a Prompt
```bash
# Using obsidian-cli
obsidian-cli print "Prompts/My_Prompts/Writing/Content/Weekly_Newsletter_Generator"
obsidian-cli print "Prompts/My_Prompts/IT/Documentation/Cyber_Security_Prompt_Library"

# Using cat/less
cat "Prompts/My_Prompts/Finance/Planning/Financial_Planner_App.md"
less "Prompts/My_Prompts/IT/Documentation/IAM_Prompt_Library.md"
```

### Create New Prompt
```bash
# Create in appropriate category
obsidian-cli create "Prompts/My_Prompts/Images/Logos/Brand_Logo_Generator"
obsidian-cli create "Prompts/My_Prompts/IT/Development/Python/Data_Analysis_Script"
obsidian-cli create "Prompts/My_Prompts/Business/Marketing/Email/Campaign_Template"
obsidian-cli create "Prompts/My_Prompts/IT/Security/Policies/Security_Policy_Template"
```

### Search Across All Prompts
```bash
# Search by keyword
obsidian-cli search-content "automation" --path "Prompts/My_Prompts"
obsidian-cli search-content "vulnerability" --path "Prompts/My_Prompts/IT/Documentation"

# Search in specific category
obsidian-cli search-content "API" --path "Prompts/My_Prompts/IT"
obsidian-cli search-content "incident" --path "Prompts/My_Prompts/IT/Documentation"
```
