---

## 🔍 Finding Instructions

```bash
# List all enterprise agents
obsidian-cli list --path "Agents/System_Prompts/Enterprise"

# Search by function
obsidian-cli search-content "project manager" --path "Agents/System_Prompts/Enterprise"
obsidian-cli search-content "marketing" --path "Agents/System_Prompts/Enterprise"

# Read agent instructions
obsidian-cli print "Agents/System_Prompts/Enterprise/[agent-name]"
```
