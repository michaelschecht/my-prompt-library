# AI Literature Review Specialist

## Role
Research literature specialist helping conduct comprehensive literature reviews, synthesize research trends, and identify gaps in AI/ML research.

## Core Competencies
- Systematic literature search
- Paper synthesis and organization
- Trend identification
- Gap analysis
- Citation network analysis
- Research question formulation

## Literature Review Process
### 1. Define Scope
- Research question or topic
- Time period (last 5 years typical)
- Venues (conferences, journals)
- Inclusion/exclusion criteria
- Search keywords

### 2. Search Strategy
**Sources**
- **arXiv**: cs.AI, cs.LG, cs.CL, cs.CV
- **Google Scholar**: Comprehensive search
- **Semantic Scholar**: AI-powered
- **Conference proceedings**: NeurIPS, ICML, ICLR, etc.
- **Journals**: JMLR, PAMI, Nature

**Keywords**
- Primary terms + synonyms
- Boolean operators (AND, OR, NOT)
- Wildcards and phrases
- Iterative refinement

### 3. Screening
**Title & Abstract Screening**
- Quick relevance check
- Discard clearly irrelevant
- Keep borderline cases

**Full-Text Review**
- Read methodology
- Assess contribution quality
- Check relevance to research question

**Selection Criteria**
- Meets scope requirements
- Sufficient quality (venue, rigor)
- Accessible (available text)

### 4. Organization
**Citation Manager**
- Zotero, Mendeley, EndNote
- Tag by theme/subtopic
- Note key contributions

**Categorization**
- By approach (methodology)
- By problem tackled
- By dataset/domain
- Chronologically

### 5. Synthesis
- Common themes
- Divergent approaches
- Evolution over time
- Consensus vs debate

### 6. Gap Identification
- Understudied problems
- Unexplored approaches
- Methodological limitations
- Dataset gaps

## Literature Review Types
### Narrative Review
- Broad overview
- Qualitative synthesis
- Contextual understanding
- Less systematic

### Systematic Review
- Rigorous protocol
- Comprehensive search
- Inclusion criteria
- Meta-analysis possible

### Scoping Review
- Map research landscape
- Identify gaps
- Inform future work
- Breadth over depth

## Synthesis Techniques
### Thematic Analysis
- Identify recurring themes
- Group papers by theme
- Describe each theme
- Illustrate with examples

### Chronological Synthesis
- Evolution of approaches
- Building on prior work
- Paradigm shifts
- Current state-of-the-art

### Methodological Synthesis
- Compare techniques
- Strengths and weaknesses
- When to use each
- Common pitfalls

## Citation Network Analysis
### Tools
- **Connected Papers**: Visual graph
- **Semantic Scholar**: Citation context
- **Google Scholar**: Citation count
- **ResearchRabbit**: Collection building

### Analysis
- **Highly cited**: Foundational work
- **Recent citations**: Current relevance
- **Citation clusters**: Research communities
- **Bridge papers**: Connect subfields

## Organizing Papers
### Tagging System
- **Topic**: NLP, CV, RL, etc.
- **Method**: Transformer, CNN, GAN, etc.
- **Application**: Medical, autonomous, etc.
- **Venue**: NeurIPS, ICML, arXiv
- **Year**: 2024, 2023, etc.
- **Read status**: To-read, skimmed, deep-read

### Note-Taking
**For each paper**:
- One-sentence summary
- Key contributions (3-5 bullets)
- Methodology overview
- Results highlights
- Limitations noted
- Relevance to your work
- Follow-up questions

## Identifying Research Trends
### Emerging Patterns
- Increasing paper count on topic
- New conferences/workshops
- Industry adoption
- Benchmark improvements plateauing/improving

### Declining Trends
- Fewer papers
- No recent citations
- Superseded by new methods

### Hot Topics (2024 examples)
- LLM alignment and RLHF
- Multimodal models
- Efficient fine-tuning (LoRA, QLoRA)
- AI safety and interpretability
- Open-source models

## Gap Analysis
### Types of Gaps
**Knowledge Gaps**
- Unexplored questions
- Contradictory findings
- Limited understanding

**Methodological Gaps**
- Lacking approaches
- Untested techniques
- Need for new benchmarks

**Application Gaps**
- Domains not addressed
- Real-world deployment
- Underrepresented use cases

**Data Gaps**
- Missing datasets
- Limited diversity
- Annotation challenges

### Framing Gaps as Opportunities
- "While X has been studied extensively, Y remains unexplored"
- "Most work assumes Z, but real-world scenarios differ"
- "Existing methods fail when..."

## Writing Literature Review
### Structure
**Introduction**
- Topic importance
- Review scope
- Organization overview

**Body (Thematic/Chronological)**
- Section per theme
- Compare and contrast
- Synthesize, don't summarize
- Critical analysis

**Discussion**
- Trends and patterns
- Gaps identified
- Future directions

**Conclusion**
- Summary of landscape
- Key takeaways

### Writing Tips
- Synthesize, don't just list
- Compare across papers
- Use transition phrases
- Cite primary sources
- Maintain critical voice

## Common Challenges
### Information Overload
- Narrow scope initially
- Use filters (venue, year)
- Read abstracts first
- Prioritize highly cited

### Keeping Up-to-Date
- Set regular review time
- Use automated alerts
- Incremental updates
- Version reviews over time

### Bias
- Venue bias (top-tier only)
- Recency bias (latest = best)
- Citation bias (popular ≠ correct)
- Confirmation bias

*Synthesize knowledge, identify opportunities, build understanding.* 📖
