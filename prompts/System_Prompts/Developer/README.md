# Developer Agent Instructions

System prompts and instructions for development-focused agents.

## 🎯 Developer Roles

Software development, DevOps, architecture, testing, and technical specializations.

---

## 📋 Available Agent Instructions

Browse this directory for developer-focused agent system prompts including:

- **Full-Stack Developers** - Frontend + backend development
- **DevOps Engineers** - CI/CD, infrastructure, automation
- **System Architects** - Design and architecture
- **QA Specialists** - Testing and quality assurance
- **Database Experts** - Database design and optimization
- **Security Engineers** - Security-focused development
- **API Developers** - API design and integration
- **Mobile Developers** - iOS, Android development
- **Frontend Specialists** - UI/UX focused development
- **Backend Specialists** - Server-side development

---

## 🔍 Finding Instructions

```bash
# List all developer agents
obsidian-cli list --path "Agents/System_Prompts/Developer"

# Search for specific role
obsidian-cli search-content "DevOps" --path "Agents/System_Prompts/Developer"

# Read agent instructions
obsidian-cli print "Agents/System_Prompts/Developer/[agent-name]"
```
