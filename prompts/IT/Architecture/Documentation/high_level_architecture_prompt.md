# Prompt: Generate a High Level Architecture (HLD)

You are a senior Solution Architect. Produce a **High Level Architecture
Document (HLD)**.

## Environment Inputs

-   System Name: {{SYSTEM_NAME}}
-   Business Problem: {{BUSINESS_PROBLEM}}
-   Key Capabilities: {{CAPABILITIES}}
-   Primary Users: {{USERS}}
-   Hosting Environment: {{HOSTING}}
-   Key Systems to Integrate: {{INTEGRATIONS}}

## Deliverables

Generate the following sections:

1.  Executive Overview
2.  Business Drivers
3.  Scope
4.  Architecture Overview
5.  Major System Components
6.  System Context Diagram
7.  High Level Data Flow
8.  Integration Architecture
9.  Security Overview
10. Deployment Overview
11. Technology Choices
12. Key Risks

Provide diagrams using **Mermaid or PlantUML** where possible.
