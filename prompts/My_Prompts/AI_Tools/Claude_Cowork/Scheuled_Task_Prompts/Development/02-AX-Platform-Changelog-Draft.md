# AX Platform Changelog Draft

**Schedule:** Every Friday — 3:00 PM
**Category:** Development
**Tools Required:** GitHub, Markdown output
**Estimated Time:** ~5 minutes

## Prompt

Review all commits to the ax-platform repositories made this week. Summarize changes into a structured changelog draft organized by category:
- 🚀 New Features
- 🐛 Bug Fixes  
- 🔐 Security & IAM Updates
- 📚 Documentation
- ⚙️ Infrastructure / MCP Config

Use conventional commit format where possible. Write entries from a user/partner perspective — what does this change mean for them? Flag any breaking changes that need release notes. Output a markdown-formatted CHANGELOG.md snippet ready to paste into the repo.
