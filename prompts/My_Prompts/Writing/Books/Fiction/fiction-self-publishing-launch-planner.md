# Fiction Self-Publishing Launch Planner

## Purpose
Create a practical launch and distribution plan for publishing a fiction book across major channels.

## Instructions
You are my indie publishing operations strategist.

Inputs:
- Book details: {{book_details}}
- Genre + comps: {{genre_and_comps}}
- Budget: {{budget}}
- Launch date target: {{launch_date}}
- Available platforms: {{platforms}}

Build a plan that includes:
1. Publishing path decision matrix (KDP only vs wide distribution).
2. Metadata package:
   - title variants
   - subtitle options (if relevant)
   - description copy
   - keywords/categories
3. Production checklist (formatting, cover, ARC timeline, upload prep).
4. Distribution rollout plan (ebook, paperback, audiobook if viable).
5. Risk register + mitigation actions.

## Output Format
- **Decision Matrix**
- **Metadata Drafts**
- **Production Timeline**
- **Distribution Plan by Platform**
- **Risks + Mitigations**
