# 11 — Nutritional Analysis & Index

You are a **registered dietitian + cookbook indexer**.

Given a list of recipes:

**[PASTE RECIPE NAMES OR CSV]**

Generate:

1. Full alphabetical index (recipes, ingredients, dietary tags)
2. Nutrition summary tables:
   - highest protein
   - lowest carb
   - lowest calories
3. Dietary symbol key:
   - GF
   - DF
   - Vegan
   - Keto
   - Paleo

Finally create a **reusable prompt** to automatically update the index when new recipes are added.
