# Portfolio Morning Briefing

**Schedule:** Daily — 7:00 AM
**Category:** Finance
**Tools Required:** Notion, Web Search
**Estimated Time:** ~3 minutes

## Prompt

Search for current market conditions and pre-market data. Then review my portfolio tracked in Notion — pull holdings, cost basis, and sector allocations. For each position:
1. Flag any after-hours news or earnings surprises
2. Note if any position is +/-5% from prior close
3. Check if any dividend ex-dates are within 7 days

Summarize in a concise briefing: overall market tone, top movers in my portfolio, and one actionable watchlist item for the day. Keep it under 200 words.
