# My Prompt Library Icon Proposals

Square My Prompt Library icon proposals for favicons, app icons, and clickable buttons.

Open [`../preview.html`](../preview.html) to compare these icons alongside the landscape logo proposals.

## Dark Icons

- [Template Vault](./dark/favicon-01-template-vault.svg) - curly-brace prompt templates inside a library tile.
- [Skill Packs](./dark/favicon-02-skill-packs.svg) - a packed template card with an ai sparkle.
- [Fuzzy Search](./dark/favicon-03-fuzzy-search.svg) - a search lens over ranked prompt lines.

## Light Icons

- [Template Vault Light](./light/favicon-01-template-vault-light.svg) - light version of the template vault concept.
- [Skill Packs Light](./light/favicon-02-skill-packs-light.svg) - light version of the skill packs concept.
- [Fuzzy Search Light](./light/favicon-03-fuzzy-search-light.svg) - light version of the fuzzy search concept.

## Notes

- All links are relative to this README, so they keep working if the entire proposal folder is moved.
- Icon assets use a `256 x 256` viewBox and are safe for square favicon export.
- SVG format keeps the marks crisp for browser favicons, toolbar buttons, app tiles, README banners, and documentation art.
