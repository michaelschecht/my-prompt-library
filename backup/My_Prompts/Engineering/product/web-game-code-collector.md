# Web Game Code Collector

**Category:** product

## Prompt

Build me a web based game. this will be a Resource collecting/building game. You will need to scour the web (and other places) for codes. If you find one, enter in the game to unlock in game resources/currency you can use to by stuff. This will allow us to generate revenue by putting codes on other websites that want traffic. You come up with a cool game design.
