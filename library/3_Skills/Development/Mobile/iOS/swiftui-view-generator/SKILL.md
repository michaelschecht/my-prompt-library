# SwiftUI View Generator

## Overview

Create SwiftUI views following MVVM architecture, using Combine for reactive programming and adhering to Apple's Human Interface Guidelines.

## Usage

```
Generate a SwiftUI view for [purpose]:
- MVVM architecture
- ViewModel with @Published properties
- Combine framework
- [Specific requirements]
```

## Examples

### Settings Screen
"Create a SwiftUI settings screen with toggle switches, navigation links, and user preferences. Use MVVM with UserDefaults persistence."

### Data List
"Build a SwiftUI List view displaying items from API with loading state, error handling, and pull-to-refresh."

### Custom Button
"Generate a reusable SwiftUI button component with customizable colors, icons, and loading state."

## Features Included

- **SwiftUI Views:** Declarative UI code
- **ViewModels:** ObservableObject with @Published
- **Combine:** Reactive data flow
- **Async/Await:** Modern concurrency
- **Accessibility:** VoiceOver support

## Output Format

1. SwiftUI View file
2. ViewModel file
3. Model/DTO (if needed)
4. Preview provider
5. Usage documentation
