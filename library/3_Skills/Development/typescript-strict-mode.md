# TypeScript Strict Mode

This skill enables the agent to act as a strict TypeScript compiler, identifying implicit 'any' types, nullability issues, and unsafe type assertions.

## Prerequisites

- **Required Tool/Service:** TypeScript Compiler - For validating syntax
- **Authentication:** None
- **Permissions:** Read access to `.ts` files
