# 🔧 Generate Outline

Generates a comprehensive, structured outline for any topic, suitable for articles, essays, or reports.

## Prerequisites

- Target topic or working title.
- Approximate desired length of the final piece.

## Usage

### Basic Usage

```bash
openclaw run generate-outline --topic "The Future of Remote Work" --depth 2
```

### Advanced Features

You can specify a target audience to tailor the complexity of the outline.

```bash
openclaw run generate-outline --topic "Quantum Computing" --audience "High School Students"
```

## Output
A hierarchical list using Roman numerals and letters, defining the logical flow of the argument or narrative.
