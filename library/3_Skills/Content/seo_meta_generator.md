# SEO Meta Generator

## Prerequisites
None.

## Usage
Provide the title and brief content of a webpage, and the skill generates optimized Meta Titles and Descriptions.

## Configuration
Specify the target keyword if applicable.

## Examples
**Input:**
Title: "Best Running Shoes 2023"
Content: "A guide to the top running shoes for marathons and daily training."
Keyword: "running shoes"

**Output:**
Meta Title: Best Running Shoes 2023 | Top Picks for Marathons & Training (58 chars)
Meta Description: Discover the best running shoes of 2023. Our comprehensive guide covers top picks for marathon runners and daily trainers to improve your performance. (152 chars)

## Advanced Usage
Generate open graph (OG) tags for social media sharing.

## Integration
Useful in CMS workflows (WordPress, Ghost).

## Troubleshooting
If descriptions exceed character limits, provide shorter content summaries.

## Best Practices
Keep titles under 60 characters and descriptions under 160 characters.

## Performance Considerations
Generates instantly based on text analysis.

## Security & Safety
Safe to use on all public content.
