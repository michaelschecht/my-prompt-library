# Modern Website Favicon Set Generator

Create **2 distinct favicon concepts** for [BRAND NAME] based on this brand direction: [BRAND STYLE / COLORS / KEYWORD VIBE].

Design goals:
- Modern, sleek, minimalist
- Simple shapes that remain legible at very small sizes
- Strong contrast on light and dark browser themes
- Avoid tiny details, thin lines, and clutter

For each concept, produce a complete favicon package using standard web formats:
- `favicon.ico` (multi-size)
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png` (180x180)
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`
- `site.webmanifest`
- Optional source: `favicon.svg`

Requirements:
- Ensure pixel clarity at 16x16 and 32x32
- Keep icon centered with balanced padding
- Use transparent background where appropriate
- Include hex color values and brief rationale for each concept

Output:
1. Two favicon design concepts (A and B) with short visual descriptions
2. Export-ready asset list with exact dimensions and file names
3. Recommended final concept and why
4. Copy/paste HTML `<link>` tags for favicon integration
