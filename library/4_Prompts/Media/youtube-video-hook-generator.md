# YouTube Video Hook Generator

## Purpose
Generate 5 engaging 15-second hooks for YouTube videos based on the topic.

## Instructions
Input the main topic or title of the YouTube video.

## Prompt

```
I am creating a YouTube video about: [INSERT VIDEO TOPIC].

Generate 5 distinct, high-retention hooks for the first 15 seconds of the video.
The hooks should:
- Immediately grab attention.
- Create a knowledge gap or ask a compelling question.
- Be spoken word (script format).

Label each hook with the 'style' of hook (e.g., The Shocking Stat, The Story, The Direct Promise).
```

## Output Format
Numbered list of scripts with style labels.

## Example

**1. The Direct Promise**
"By the end of this video, you'll know exactly how to double your coding speed without learning a new language."
