Research and compare options before making a technical decision.

## Prompt

```
Research the top options for [DECISION]. For each option, find: current maintenance status, performance benchmarks, pricing, notable adopters. Produce a comparison table and a recommendation with rationale.
```

## Usage Notes

- Replace `[DECISION]` with your evaluation subject (e.g., choosing a vector database)
- Output includes a comparison table and a reasoned recommendation
- Useful for architecture decisions, vendor selection, and tooling choices
- Ask for the recommendation to be opinionated, not wishy-washy
