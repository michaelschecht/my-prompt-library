Generate thorough inline documentation for an existing module.

## Prompt

```
Read [FILE]. Generate comprehensive documentation including: module overview, all exported functions with parameter types and return values, usage examples for each function, and any important caveats. Output as JSDoc/docstring comments inline, plus a README section.
```

## Usage Notes

- Replace `[FILE]` with the module to document
- JSDoc/docstrings enable IDE tooltip support and automated doc generation
- Usage examples are required — not optional
- The README section is ready to paste into your project docs
