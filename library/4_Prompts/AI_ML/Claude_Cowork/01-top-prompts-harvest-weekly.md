# Top Prompts Harvest — Weekly

**Schedule:** Every Wednesday — 8:00 AM
**Category:** AI Prompts
**Tools Required:** Web Search, Web Fetch
**Estimated Time:** ~10 minutes

## Prompt

Search for the most popular, most-upvoted, or most-shared AI prompts published this week. Check: PromptHero, PromptBase, r/ChatGPT, r/ClaudeAI, r/PromptEngineering, FlowGPT, ShareGPT, and any curated GitHub "awesome-prompts" repos with recent commits.

Collect the top 10 most notable prompts. For each:
1. The full prompt text (fetch if behind a link)
2. Source platform and engagement metric (upvotes, saves, views)
3. Category tag: [productivity] [coding] [creative writing] [roleplay] [research] [image gen] [business] [other]
4. A one-line "why it works" annotation — what technique makes it effective?
5. Which model it was designed for (if specified)

Save all 10 as individual files to ~/prompts/popular/[YYYY-MM-DD]-[slug].md. Append a summary table row to ~/prompts/popular/index.md with: date, title, source, category, and engagement score.
