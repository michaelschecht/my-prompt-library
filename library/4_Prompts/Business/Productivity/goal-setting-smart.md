# SMART Goal Setting Framework

## Purpose
Set clear, achievable goals using the SMART framework (Specific, Measurable, Achievable, Relevant, Time-bound).

## Instructions

I want you to help me implement the SMART Goal Setting Framework for my work.

**Current situation:**
- **Work type:** [Your job/role]
- **Main challenges:** [Current productivity issues]
- **Goals:** [What you want to achieve]

Provide a customized system including:
- Step-by-step implementation
- Templates and checklists
- Common pitfalls to avoid
- Metrics to track success

## Tips for Use

- Start with small changes and build gradually
- Track what works and adjust based on results
- Consistency beats perfection - implement imperfectly but consistently
