# Startup Pitch Deck Outline

## Purpose
Generate a standard 10-15 slide pitch deck outline based on a company description.

## Instructions
Provide a brief description of the startup, its product, and target market.

## Prompt

```
I need a comprehensive pitch deck outline for my startup.
Here is the description of the company: [INSERT STARTUP DESCRIPTION]

Please generate a 10-12 slide outline following standard Silicon Valley best practices (e.g., Sequoia Capital format). For each slide, provide:
1. The slide title (e.g., "The Problem").
2. 2-3 bullet points detailing exactly what information should go on that slide based on my startup description.
```

## Output Format
Numbered list of slides with bulleted details.

## Example

**Slide 1: Problem**
- Highlight the inefficiency of current manual data entry.
- Show the financial cost to enterprise companies.
