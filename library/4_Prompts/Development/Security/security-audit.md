# Network Security Audit Checklist

Use this prompt to generate a comprehensive security audit checklist for a corporate network.

## Parameters
- **Network Size:** [Small | Medium | Large]
- **Infrastructure:** [On-premise | Cloud | Hybrid]
- **Compliance Requirements:** [SOC2 | HIPAA | GDPR | None]

## Prompt
Generate a detailed network security audit checklist for a [NETWORK SIZE] network with [INFRASTRUCTURE] infrastructure. Ensure the checklist covers:
1. Physical Security
2. Network Perimeter Defense
3. Internal Network Segmentation
4. Access Control and Identity Management
5. Data Encryption (at rest and in transit)
6. Logging and Monitoring
7. Incident Response Readiness
