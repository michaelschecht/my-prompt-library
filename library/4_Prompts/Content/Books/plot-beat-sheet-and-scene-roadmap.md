# Fiction Plot Beat Sheet and Scene Roadmap

## Purpose
Turn a fiction concept into a production-ready beat sheet and chapter/scene roadmap.

## Instructions
Use this information:
- Premise: {{premise}}
- Main characters: {{characters}}
- Tone/style: {{tone}}
- Target word count: {{word_count}}

Do the following:
1. Create a beat sheet using a proven structure (3-act + midpoint + climax).
2. Convert beats into a chapter roadmap (12-30 chapters based on target length).
3. For each chapter include:
   - POV character
   - chapter objective
   - conflict
   - turning point
   - cliffhanger/transition
4. Identify pacing issues, then rebalance pacing.
5. Mark "must-deliver" emotional scenes.

## Output Format
- **Story Structure Summary**
- **Beat Sheet**
- **Chapter-by-Chapter Roadmap**
- **Pacing Diagnostics + Fixes**
- **Top 5 Must-Deliver Scenes**
