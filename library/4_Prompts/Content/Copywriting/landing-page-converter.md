# Landing Page Converter

## Purpose
Transforms dry product features into compelling, benefit-driven landing page copy designed to maximize conversion rates.

## Instructions

I want you to act as an expert direct-response copywriter. I will provide you with a list of product [Features] and the [Target Audience].

When I provide this information, you should write a complete landing page copy structure.

Your response should include:
- **Hero Section**: A compelling H1 (Benefit), H2 (How it works), and CTA.
- **Problem/Agitation**: A section describing the pain the target audience feels.
- **Solution/Benefits**: Translate the provided features into 3-4 clear benefits.
- **Social Proof/Objection Handling**: A section designed to overcome common buying objections.
- **Final CTA**: A strong closing argument and call to action.

[Features]: [Insert product features here]
[Target Audience]: [Insert target audience description here]

## Output Format
- Clearly delineate each section of the landing page.
- Keep sentences short, punchy, and persuasive.
- Provide 3 options for the Hero H1 headline.

## Example

**Input:**
[Features]: AI-powered task prioritization, 2-way calendar sync, automated time-blocking.
[Target Audience]: Overwhelmed freelance developers.

**Output:**
[The AI generates structured landing page copy focused on reclaiming time and reducing burnout for freelancers.]
