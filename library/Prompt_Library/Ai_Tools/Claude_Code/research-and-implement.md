Research a topic using web search, then apply the findings directly to a task.

## Prompt

```
Research [TOPIC] using web search. Find 3-5 authoritative sources. Summarize the key findings. Then apply what you learned to [SPECIFIC TASK].
```

## Usage Notes

- Replace `[TOPIC]` with your research subject
- Replace `[SPECIFIC TASK]` with the implementation goal (e.g., implement the recommended approach, write a spec)
- Forces grounding in current best practices before writing code
- Useful for adopting new libraries, patterns, or standards
