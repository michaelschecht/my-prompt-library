Look up current API documentation before writing code to avoid outdated patterns.

## Prompt

```
Before writing any code for [LIBRARY/API], search the web for its latest documentation. Identify the current version, any breaking changes from older versions, and the recommended patterns. Then complete: [CODING TASK].
```

## Usage Notes

- Replace `[LIBRARY/API]` with the package or service
- Replace `[CODING TASK]` with what you want built
- Prevents use of deprecated or removed APIs
- Critical for fast-moving libraries like AI SDKs, React, etc.
