# Earnings Calendar Alert

**Schedule:** Every Sunday — 6:00 PM
**Category:** Finance
**Tools Required:** Notion, Web Search
**Estimated Time:** ~6 minutes

## Prompt

Search for the upcoming week's earnings calendar. Cross-reference against my Notion portfolio holdings and watchlist. For each company reporting this week that I hold or watch:
1. Report date + time (pre/post market)
2. Analyst consensus EPS and revenue estimates
3. Key metric to watch (e.g. cloud revenue growth, margins, guidance)
4. Historical beat/miss rate over last 4 quarters

Format as a clean weekly earnings preview table and add a "📅 Earnings This Week" note to relevant Notion entries. Flag any high-IV options plays if applicable.
