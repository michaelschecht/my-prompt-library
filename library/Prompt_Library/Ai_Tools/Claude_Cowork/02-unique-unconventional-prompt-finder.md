# Unique & Unconventional Prompt Finder

**Schedule:** Every Friday — 9:00 AM
**Category:** AI Prompts
**Tools Required:** Web Search, Web Fetch
**Estimated Time:** ~10 minutes

## Prompt

Search specifically for AI prompts that are creative, unusual, philosophically interesting, or represent novel prompting techniques — NOT standard productivity prompts. Sources to mine: LessWrong, Alignment Forum, r/PromptEngineering deep threads, Gwern.net, Simon Willison's blog, Ethan Mollick's posts, and niche AI Discord servers mentioned in articles.

Look for prompts that use: unconventional framing, meta-cognitive techniques, adversarial setups, persona paradoxes, structured chain-of-thought innovations, constraint-based creativity, or prompts that explore model behavior in interesting ways.

For each of the top 5 unique finds:
1. Full prompt text
2. What makes it conceptually different from standard prompts
3. Suggested use case or experiment to run with it
4. Difficulty/complexity rating: [Beginner / Intermediate / Advanced / Research-grade]
5. Related technique or paper if applicable

Save to ~/prompts/unique/[YYYY-MM-DD]-[slug].md. Tag with [technique type]. Append to ~/prompts/unique/index.md.
