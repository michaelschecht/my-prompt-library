# Global Cuisine Deep Dive

**Schedule:** Every Tuesday — 8:00 AM
**Category:** Recipes
**Tools Required:** Web Search, Web Fetch
**Estimated Time:** ~10 minutes

## Prompt

Each week focus on one underrepresented or regional cuisine. Rotate through: West African, Peruvian, Georgian (Caucasus), Yemeni, Oaxacan, Sichuan, Filipino, Ethiopian, Basque, Sri Lankan, Palestinian, and others. This week's cuisine: [AUTO-ROTATE or specify].

Search for authentic recipe sources — prioritize food writers or home cooks from that culture over mainstream sites. Find:
1. One "gateway dish" — the most accessible entry point to the cuisine
2. One "cornerstone dish" — a defining recipe that serious cooks should know
3. One pantry/ingredient guide — what staples to stock for this cuisine

Fetch all three recipes in full. For each include: cultural context (2–3 sentences), ingredient substitutions for hard-to-find items, and sourcing notes. Save to ~/recipes/global/[cuisine-name]/[dish-slug].md. Update ~/recipes/global/index.md with the new entries.
