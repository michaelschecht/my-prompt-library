# Image Generation Prompt Library Builder

**Schedule:** Every Thursday — 8:30 AM
**Category:** AI Prompts
**Tools Required:** Web Search, Web Fetch
**Estimated Time:** ~9 minutes

## Prompt

Search for high-performing image generation prompts for Midjourney, DALL-E, Flux, and Stable Diffusion. Check PromptHero, Civitai, Lexica.art, and r/midjourney top posts from this week.

Focus on:
1. Prompts producing photorealistic food photography (relevant for FlavorAtlas)
2. Prompts for editorial / tech / startup visual styles (relevant for AX Platform marketing)
3. Prompts using innovative style or lighting techniques worth cataloging
4. Any "prompt formulas" — reusable structural templates with fill-in-the-blank variables

For each prompt collected:
- Full prompt text with any negative prompts
- Target model and recommended parameters (steps, CFG, aspect ratio)
- Example output description or link if available
- Category: [food] [tech/corporate] [portrait] [landscape] [abstract] [product]

Save to ~/prompts/image-gen/[model]/[YYYY-MM-DD]-[slug].md. Update ~/prompts/image-gen/index.md with new entries grouped by category.
