# SEO Audit Plan (Based on Google Search Essentials)

**Prompt:**
"Perform an SEO audit for [Website] using Google Search Essentials.
Include:
1. Technical SEO checklist (indexing, crawlability, Core Web Vitals).
2. On-page optimization analysis (titles, headings, schema).
3. Content quality review (E-E-A-T signals).
4. Backlink profile assessment and risk.
5. Prioritized fixes with expected impact.

Deliver an audit summary and action backlog."
