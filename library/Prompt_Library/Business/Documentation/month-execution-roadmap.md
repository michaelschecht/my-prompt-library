# 12-Month Execution Roadmap

## Objective
Create a disciplined 12-month roadmap for **[BUSINESS]** from **[CURRENT STAGE]** to **[TARGET MILESTONE]**.

## Inputs
- Current stage: [STAGE]
- Target milestone: [MILESTONE]
- Team/resources: [CONSTRAINTS]
- Budget envelope: [$X]

## Analysis Instructions
Plan:
- Quarterly goals and measurable outcomes
- Functional priorities (product, marketing, sales, operations)
- Budget/resource allocation by quarter
- Dependencies and critical path
- Decision gates and pivot criteria

## Required Output
1. Quarter-by-quarter roadmap
2. KPI targets by function
3. Ownership/accountability cadence
4. Top blockers and mitigation plan
5. Weekly operating rhythm
