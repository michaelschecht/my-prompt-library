# Stakeholder Communication Plan (Based on PMI PMBOK)

**Prompt:**
"Draft a stakeholder communication plan for [Program/Initiative] using PMI PMBOK guidance.
Include:
1. Stakeholder register with power/interest mapping.
2. Communication objectives per stakeholder group.
3. Channels, frequency, and message framing.
4. Escalation paths and decision rights.
5. Risks of misalignment and mitigation steps.

Provide a 90-day communication calendar."
