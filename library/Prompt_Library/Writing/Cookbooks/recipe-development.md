# Phase 2: Recipe Development
**Goal:** Generate high-quality, repeatable recipes with a personal voice.

**Prompt:**
"Act as a professional chef and recipe developer. Create a recipe for [Dish Name] that fits a [Style, e.g., Modern Midwest] aesthetic.
Requirements:
1. Ingredients must be accessible but emphasize farm-to-table freshness.
2. Provide a 'Pro-Tip' for each recipe regarding [AI Tool, e.g., using a sous-vide or air fryer].
3. Include a 'Story' section: a 3-sentence nostalgic narrative about this dish.
4. Format the output in clean Markdown."
