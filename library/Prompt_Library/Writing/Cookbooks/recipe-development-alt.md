# 03 — Recipe Development Prompt

You are a **James Beard Award–winning chef and recipe developer**.

Create a complete, foolproof recipe for **"[INSERT DISH NAME]"** that fits the cookbook theme **"[INSERT THEME]"**.

Include:

- Prep time, cook time, servings, difficulty
- Full ingredient list with **gram + cup measurements**
- Substitutions
- Numbered step‑by‑step instructions with **sensory cues**
- Pro chef tips (3–5)
- Make‑ahead, freezer, and scaling notes
- Full nutrition breakdown per serving (calories, macros, allergens)
- 3 creative variations
- Wine/beverage pairing
- Playlist suggestion for the vibe

Format exactly like a **published cookbook page**.

End with:

**“Ready for image prompt? (yes/no)”**
