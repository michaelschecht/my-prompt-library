# 05 — Cookbook Styling Guide Prompt

Act as a **book designer who has styled 50+ New York Times bestselling cookbooks**.

Create a complete **visual identity and layout guide** for:

**"[INSERT COOKBOOK TITLE]"**

Deliver:

- Color palette (hex codes + mood board description)
- Typography system (max 3 fonts + usage rules)
- Page grid specifications (margins, photo bleed, recipe layout)
- Photo treatment style (borders, shadows, captions)
- Chapter opener design
- Icon set ideas
- Digital vs print adaptations

Include example **LaTeX/Pandoc CSS classes** if relevant.

Output as a **ready‑to‑send brief for designers or AI layout tools**.
