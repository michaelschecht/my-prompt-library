# Fiction Line Edit and Beta Feedback Synthesizer

## Purpose
Transform messy draft feedback into a prioritized revision plan with concrete line-level and structural fixes.

## Instructions
Act as a fiction editor and beta feedback analyst.

Inputs:
- Manuscript excerpt/full draft: {{manuscript_text}}
- Beta feedback notes: {{beta_feedback}}
- Author goals (tone, pacing, audience): {{author_goals}}

Do the following:
1. Cluster feedback into themes (pacing, clarity, character, stakes, prose, continuity).
2. Separate signal from noise:
   - high-confidence issues (multiple readers)
   - subjective preferences (single-reader taste)
3. Provide line-edit examples (before/after) for at least 8 weak passages.
4. Build a revision sequence from highest impact to lowest.
5. Create a final polish checklist before publication.

## Output Format
- **Feedback Theme Map**
- **Signal vs Noise Analysis**
- **Line Edit Examples (Before/After)**
- **Revision Roadmap (Sprint-based)**
- **Final Polish Checklist**
