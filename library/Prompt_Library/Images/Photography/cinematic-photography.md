# Cinematic Photography

## Purpose
Generate a Midjourney image prompt that replicates the visual aesthetics of professional filmmaking, incorporating sophisticated elements like professional lighting, shallow depth of field, and specific color grading.

## Instructions
Create an image prompt using the "Cinematic Photography" style. The prompt should incorporate keywords like "movie still", "anamorphic lens", "film grain", and "color grading". It should also specify technical details such as "anamorphic bokeh", "wide aspect ratio", and "filmic color science". Describe a compelling scene, setting, and mood to anchor the cinematic elements. Include technical parameters like aspect ratio (`--ar 21:9`) at the end.

## Output Format
A single, highly detailed text prompt ready to be pasted directly into Midjourney, starting with the core subject and ending with the technical modifiers and parameters. No conversational text.

## Example
**User:** (Sends the prompt above)

**AI:**
Cinematic movie still, lone figure silhouetted walking down foggy alleyway at night, volumetric light beams cutting through mist, shot on Arri Alexa with anamorphic 50mm lens, 2.39:1 aspect ratio, teal and orange color grading, film grain texture, Blade Runner atmospheric mood, dramatic chiaroscuro lighting, Hollywood cinematography --ar 21:9
