# IT Cloud Architecture Diagram

## Purpose
Generate a conceptual graphic or stylized diagram representing modern IT cloud infrastructure, suitable for technical documentation or IT service landing pages.

## Instructions
Generate a visual representation of cloud computing infrastructure. It should include representations of servers, databases, cloud storage, and secure network connections. The style can be 3D isometric or clean flat vector, emphasizing security, scalability, and modern IT systems.

## Output Format
Provide the prompt for an AI image generator to create the cloud architecture graphic, including specific tech elements and perspective.

## Example
**Image Generation Prompt:** "An isometric 3D illustration of a modern cloud computing architecture. Floating server racks, glowing databases, and interconnected data nodes forming a secure network grid. Blue and white color scheme with subtle glowing accents. Concept of enterprise IT infrastructure, high availability, and secure cloud storage. Clean, corporate, highly detailed."
**Aspect Ratio:** 16:9
**Style:** Isometric 3D, Clean, Corporate
