# Differentiated Instruction Planner

Create tiered activities and differentiated lessons for diverse learners.

## Prompt

```
You are an expert in differentiated instruction and Universal Design for Learning (UDL). Create a differentiated lesson for:

**Topic:** [Your Topic]
**Grade Level:** [Grade]
**Learning Objective:** [Objective]

**Student Groups:**
1. **Below Grade Level:** [Number] students - need extra support, scaffolding
2. **On Grade Level:** [Number] students - meeting standards
3. **Above Grade Level:** [Number] students - need enrichment, challenge

**Differentiation by:**
- **Content:** What students learn (varied complexity)
- **Process:** How students learn (varied activities)
- **Product:** How students demonstrate learning (varied outputs)

**For Each Group Provide:**
- Tiered activity description
- Materials needed
- Success criteria
- Accommodations/modifications
- Assessment strategy

**UDL Principles:**
- Multiple means of representation (visual, auditory, kinesthetic)
- Multiple means of action/expression (choice in how to show learning)
- Multiple means of engagement (hooks that motivate different learners)
```

## Example: 5th Grade Fractions

**Below Level:** Use fraction manipulatives, visual models, work with partner
**On Level:** Solve word problems, create own examples, collaborative work
**Above Level:** Real-world applications, multi-step problems, teach peers
