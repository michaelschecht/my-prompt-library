# Assessment & Rubric Designer

Generate formative and summative assessments with clear rubrics.

## Prompt

```
Create a comprehensive assessment for:

**Subject:** [Subject]
**Grade Level:** [Grade]
**Topic/Unit:** [Topic]
**Type:** [Formative/Summative]
**Format:** [Multiple Choice/Essay/Project/Performance/etc.]

**Include:**

1. **Learning Objectives Being Assessed**
   - List 3-5 specific objectives

2. **Assessment Items**
   - [Number] questions/tasks
   - Mix of difficulty levels (Bloom's Taxonomy)
   - Clear, grade-appropriate language

3. **Answer Key** (for objective items)

4. **Scoring Rubric** (for subjective items)
   Create a 4-point rubric with:
   - Advanced (4): Exceeds expectations
   - Proficient (3): Meets expectations  
   - Developing (2): Approaching expectations
   - Beginning (1): Needs support

5. **Accommodations**
   - For students with IEPs
   - For English Language Learners
   - For students who finish early

6. **Time Estimate**
```

## Assessment Types

**Formative (During Learning):**
- Exit tickets
- Think-pair-share
- Kahoot/Quizizz
- Quick writes

**Summative (End of Unit):**
- Unit tests
- Projects
- Presentations
- Portfolios
