# Testing Strategy (Based on ISTQB and Martin Fowler)

**Prompt:**
"Create a testing strategy for [Product] using ISTQB principles and Martin Fowler’s testing pyramid.
Include:
1. Test scope by layer (unit, integration, end-to-end).
2. Coverage targets and risk-based prioritization.
3. Test data management and environment strategy.
4. CI/CD gating rules and quality metrics.
5. Defect triage workflow.

Provide a phased rollout plan."
