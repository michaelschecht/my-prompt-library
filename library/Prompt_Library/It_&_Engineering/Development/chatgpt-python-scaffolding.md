# ChatGPT Python Project Scaffolding

## Purpose
Use this prompt to generate a robust and standard Python project structure, complete with essential configuration files like `setup.py`, `requirements.txt`, and basic test configurations.

## Instructions
Act as an expert Python software engineer. I am starting a new Python project to [Describe the purpose of your project, e.g., build a REST API using FastAPI]. Please provide a complete, robust project directory structure and scaffolding.

Include the following components:
1.  **Directory Structure:** A tree representation of the recommended folder layout.
2.  **Configuration Files:** Provide the standard content for files like `requirements.txt` (or `pyproject.toml`), `.gitignore`, and a basic `README.md`.
3.  **Core Application Logic:** A simple "Hello World" or initial entry point script in the main application directory.
4.  **Testing Setup:** A basic test file using `pytest` to ensure tests are configured correctly.

Explain briefly why you chose this specific structure and best practices applied.

## Output Format
A markdown response featuring directory tree structures and separate code blocks for each core file generated.

## Example
Act as an expert Python software engineer. I am starting a new Python project to build a command-line tool for parsing CSV data. Please provide a complete, robust project directory structure and scaffolding. Includes recommendations for arg parsing libraries, a basic pyproject.toml and a test structure with pytest.
