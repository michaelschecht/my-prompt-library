# Incident Postmortem (Based on Google SRE and Etsy)

**Prompt:**
"Write a blameless incident postmortem for [Incident] using Google SRE and Etsy guidelines.
Include:
1. Timeline with detection, mitigation, and resolution.
2. Impact analysis (SLO breach, customers affected).
3. Root cause and contributing factors.
4. Action items with owners and due dates.
5. Lessons learned and prevention strategies.

Provide both executive summary and detailed sections."
