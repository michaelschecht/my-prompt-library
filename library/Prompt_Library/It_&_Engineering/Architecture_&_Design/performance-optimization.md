# Performance Optimization Plan (Based on Brendan Gregg’s USE Method)

**Prompt:**
"Diagnose and optimize [System/Service] performance using Brendan Gregg’s USE Method.
Include:
1. Resource utilization checklist (CPU, memory, storage, network).
2. Saturation and error indicators.
3. Profiling strategy (sampling, tracing) with tools.
4. Bottleneck hypotheses and experiments.
5. Optimization roadmap with expected latency/throughput gains.

Deliver a structured plan with metrics to validate improvements."
