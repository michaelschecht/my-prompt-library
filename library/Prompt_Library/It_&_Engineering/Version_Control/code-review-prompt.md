# Code Review Protocol (Based on Google Engineering Practices)

**Prompt:**
"Create a rigorous code review checklist for [Language/Stack] based on Google Engineering Practices.
Include:
1. Correctness, readability, maintainability, and test coverage criteria.
2. Security checks aligned with OWASP Top 10.
3. Performance anti-patterns and concurrency hazards.
4. Suggested review workflow and SLAs.
5. Example review comments for common issues.

Output as a checklist plus reviewer guide."
