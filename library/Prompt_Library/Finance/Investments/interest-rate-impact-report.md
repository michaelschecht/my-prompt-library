# Interest Rate Impact Report

## Objective
Assess how current/expected rate moves should change portfolio positioning.

## Inputs
- Macro horizon: [1M / 3M / 6M]
- Base view: [RATES UP / PEAKING / DOWN]
- Portfolio style: [RISK-ON / BALANCED / DEFENSIVE]

## Analysis Instructions
Evaluate rate sensitivity across:
- Equities (growth/value)
- Bonds (short/intermediate/long duration)
- Real-estate proxies
- Gold/commodities
- USD and broader risk assets

## Required Output
1. Rate regime summary
2. Asset winners/losers in this regime
3. Tactical allocation tilts for next 1–3 months
4. Top risks if rate expectations shift
5. Decision checklist for exposure adjustments
