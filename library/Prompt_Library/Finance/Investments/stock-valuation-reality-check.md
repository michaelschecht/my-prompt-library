# Stock Valuation Reality Check

## Objective
Decide whether **[TICKER]** is overvalued, fairly valued, or undervalued for **[TIME HORIZON]**.

## Inputs
- Ticker: [TICKER]
- Horizon: [3M / 12M / 3Y]
- Style: [VALUE / GROWTH / BLEND]

## Analysis Instructions
Evaluate:
- Revenue/earnings growth quality
- Margin durability and cash-flow strength
- Relative valuation vs peers
- Historical valuation vs current range
- Narrative risk (assumptions already priced in)

## Required Output
1. Valuation verdict (Over / Fair / Under)
2. Fair value range + confidence level
3. Conditions required for upside thesis
4. Risks that would break the thesis
5. Decision: Buy / Wait / Avoid + rationale
