# Revenue Growth Projection

Projects future revenue using historical data and specified growth assumptions.

---

## Prompt

Project revenue for the next [N_PERIODS] periods based on the following historical data and assumptions.\n\nHistorical Data:\n[HISTORICAL REVENUE DATA]\n\nAssumptions:\n[GROWTH RATE / TREND ASSUMPTIONS]\n[SEASONALITY FACTORS (If any)]\n\n1. Calculate the projected revenue for each of the next [N_PERIODS].\n2. Show the absolute dollar increase and percentage increase period-over-period.\n3. Identify any major risks to achieving this forecast.\n\nFormat the output as a clear markdown table followed by brief bullet points for risks.

---

## Output Structure

1. Projection Table (Period, Projected Revenue, $ Increase, % Increase)\n2. Summary of Key Assumptions Used\n3. Forecast Risks (2-3 bullet points)
