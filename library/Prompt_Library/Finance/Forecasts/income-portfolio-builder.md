# Income Portfolio Builder

## Objective
Design an income-focused portfolio for **[RISK LEVEL]** targeting **[YIELD TARGET]%** annual yield.

## Inputs
- Risk level: [CONSERVATIVE / MODERATE / AGGRESSIVE]
- Yield target: [X%]
- Time horizon: [YEARS]
- Constraints: [NO OPTIONS / ETF-ONLY / TAXABLE / IRA]

## Analysis Instructions
Build a diversified income model using:
- Dividend equities
- Bond ETFs / fixed income
- REITs
- Cash equivalents

Evaluate:
- Yield sustainability vs yield-trap risk
- Drawdown resilience in weak markets
- Rebalancing logic and diversification quality
- Tax-efficiency at a high level

## Required Output
1. Suggested allocation by sleeve (%)
2. Expected yield range and key assumptions
3. Top risks and mitigation plan
4. Buy/watchlist candidates by sleeve
5. Ongoing monitoring checklist
