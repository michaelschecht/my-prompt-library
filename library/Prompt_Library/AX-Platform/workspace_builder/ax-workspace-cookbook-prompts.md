# AX Workspace Cookbook Prompts

**Skill:** ax-workspace-cookbook\
**Skill Link:**
<https://github.com/AX-MCP/AX-CommunityWorkspaces/tree/main/ax-workspace-cookbooks/claude_skills/ax-workspace-cookbook>

This document contains example prompts users can run with the
**ax-workspace-cookbook** skill to generate all artifacts required for a
new AX workspace:

- agents.md
- tools.md
- workflow.md
- workspace_setup.md

Each prompt explicitly references the skill name so it can be invoked
directly.

---

**Example Prompts**

## 1. Enterprise SOC Workspace

Use the **ax-workspace-cookbook** skill to build a workspace for
enterprise SOC operations that triages SIEM alerts, investigates IAM
anomalies, creates incident tickets, and escalates high‑risk threats to
human analysts.

## 2. Identity & Access Governance

Use the **ax-workspace-cookbook** skill to create a cookbook for an
enterprise identity and access governance workspace that automates
access reviews, detects privilege escalation, and produces compliance
reports for auditors.

## 3. DevSecOps CI/CD Pipeline

Use the **ax-workspace-cookbook** skill to design an agent team
workspace that manages DevSecOps pipelines, performs automated code
security scans, monitors deployments, and coordinates rollback actions
if vulnerabilities are detected.

## 4. Cloud Security Monitoring

Use the **ax-workspace-cookbook** skill to build a workspace for cloud
security monitoring that analyzes AWS and GCP logs, detects
misconfigurations, correlates IAM activity with alerts, and generates
remediation tasks.

## 5. Customer Support Automation

Use the **ax-workspace-cookbook** skill to create a cookbook for a
customer support automation workspace that triages support tickets,
searches a knowledge base, escalates complex issues to human agents, and
tracks SLA performance.

## 6. Agile Scrum Product Team

Use the **ax-workspace-cookbook** skill to build a workspace where AI
agents collaborate as a scrum team managing sprint planning, backlog
grooming, bug triage, release readiness, and engineering coordination.

## 7. Financial Research Workspace

Use the **ax-workspace-cookbook** skill to create a cookbook for a
financial research workspace where agents analyze market data, summarize
earnings reports, detect anomalies, and generate investment insights.

## 8. AI Developer Community Workspace

Use the **ax-workspace-cookbook** skill to design a workspace for a
community of AI developers collaborating on MCP projects, sharing
experiments, reviewing agent workflows, and publishing open‑source
tools.

## 9. Fantasy Sports Analytics Workspace

Use the **ax-workspace-cookbook** skill to build a niche workspace where
agents collaborate to manage fantasy football teams by analyzing player
statistics, injury reports, matchup data, and recommending optimal
lineups.

## 10. Podcast & Content Production Workspace

Use the **ax-workspace-cookbook** skill to create a cookbook for a
podcast production workspace where agents research topics, generate
episode outlines, coordinate guest outreach, and publish show notes.

---

# How to Use

1. Copy one of the prompts above.
2. Paste it into Claude where the **ax-workspace-cookbook** skill is
    available.
3. The skill will generate four implementation-ready files:

- agents.md
- tools.md
- workflow.md
- workspace_setup.md

These artifacts can then be used directly to create a new AX workspace
and configure the agents and integrations required.
