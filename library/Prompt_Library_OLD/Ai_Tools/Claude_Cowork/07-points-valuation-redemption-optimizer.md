# Points Valuation & Redemption Optimizer

**Schedule:** Every Sunday — 9:00 AM
**Category:** Rewards
**Tools Required:** Web Search
**Estimated Time:** ~6 minutes

## Prompt

Search for the latest points and miles valuations from The Points Guy, NerdWallet, and Upgraded Points. Then build a personal redemption strategy snapshot:

1. Pull current CPP (cents per point) valuations for: Chase UR, Amex MR, Citi TY, Capital One, Bilt, United, Delta, Southwest, Hyatt, Marriott, Hilton
2. Flag any programs where valuations have changed more than 0.2 CPP since last month
3. Identify my highest-value redemption opportunity right now based on balances: [INSERT CURRENT BALANCES]
4. Surface any upcoming devaluations or program changes that should trigger "burn before devalue" action

Output as a ranked "Redeem Now vs. Hold" table. Include a recommended burn target for any currency currently above its peak valuation. Add a note on any currencies worth accumulating vs. spending this quarter.
