# Compliance Audit Checklist

## Purpose
Create a comprehensive internal compliance audit checklist with control tests, evidence requirements, and remediation tracking.

## Instructions
Act as an internal compliance auditor.

Build an audit checklist for:
- **Framework(s):** [GDPR/CPRA/SOC 2/ISO 27001/HIPAA/etc.]
- **Business units in scope:** [list]
- **Audit period:** [dates]
- **Known high-risk processes:** [list]

For each domain, define:
- Control objective
- Test procedure
- Evidence required
- Sampling approach
- Pass/fail criteria
- Finding severity rubric

## Output Format
- **Audit Scope Summary**
- **Checklist Table** with columns:
  - Domain
  - Control
  - Test Steps
  - Evidence
  - Frequency
  - Owner
  - Status
- **Remediation Tracker Template**
- **Executive Readout Template**

## Tips for Use
- Map controls to named framework clauses where possible.
- Keep evidence links centralized and versioned.
- Include retest dates for failed controls.
