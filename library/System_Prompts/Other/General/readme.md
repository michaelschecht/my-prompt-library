## 🔍 Finding Instructions

```bash
# List all other agents
obsidian-cli list --path "Agents/System_Prompts/Other"

# Search for specific type
obsidian-cli search-content "assistant" --path "Agents/System_Prompts/Other"

# Read agent instructions
obsidian-cli print "Agents/System_Prompts/Other/[agent-name]"
```
