# Git Automation

This skill streamlines your Git version control workflows.

## Capabilities

*   **Commit Message Generation:** Automatically generates descriptive and conventional commit messages based on staged changes.
*   **Merge Conflict Resolution:** Provides guidance and automated scripts to help identify and resolve complex merge conflicts.
*   **Branch Management:** Automates creating, deleting, and merging branches based on standard branching strategies (e.g., GitFlow).
*   **Repository Analysis:** Analyzes commit history to generate changelogs or identify active contributors.
