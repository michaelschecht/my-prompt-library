# Data Cleaning

This skill helps you prepare raw data for analysis.

## Common Tasks

*   **Handling Missing Data:** Imputing missing values using mean, median, mode, or more advanced techniques like KNN imputation.
*   **Removing Duplicates:** Identifying and eliminating exact or fuzzy duplicate records.
*   **Formatting and Standardization:** Ensuring consistent data formats (e.g., dates, phone numbers, text casing).
*   **Outlier Detection:** Identifying anomalous data points that may skew analysis results.
