## 🔍 Finding Instructions

```bash

# Search by domain
obsidian-cli search-content "financial" --path "Agents/System_Prompts/Specialized"
obsidian-cli search-content "security" --path "Agents/System_Prompts/Specialized"

# Read agent instructions
obsidian-cli print "Agents/System_Prompts/Specialized/[agent-name]"
```
