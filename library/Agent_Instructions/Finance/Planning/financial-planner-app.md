# Financial Planning Report Builder

## Objective
Generate a practical financial planning report to support purchasing and trading decisions for **[PERSON OR PORTFOLIO NAME]**.

## Inputs
- Primary goals: [CAPITAL GROWTH / INCOME / PRESERVATION]
- Time horizon: [SHORT / MEDIUM / LONG]
- Risk tolerance: [LOW / MEDIUM / HIGH]
- Constraints: [CASH NEEDS, TAX, ACCOUNT TYPE, ETHICAL FILTERS]

## Analysis Instructions
Create a report that includes:
- Current portfolio snapshot and concentration risks
- Cash flow and liquidity readiness for near-term opportunities
- Position-level opportunity/risk summary
- Buy/watch/trim candidates with reasoning
- Rebalancing and risk-control priorities

## Required Output
1. Executive summary with current stance
2. Portfolio health scorecard (strengths + vulnerabilities)
3. Priority action list for next 7/30/90 days
4. Decision rules for new purchases and exits
5. Monitoring checklist with triggers that require action

## Guardrail
Provide educational analytics and scenario planning only; do not present content as personalized financial advice.
