# Cloud Hosting Company Logo

## Purpose
Design a solid, reliable, and modern logo for a B2B cloud hosting or IT infrastructure provider.

## Instructions
Create a logo that blends elements of cloud computing (e.g., stylized clouds, servers, data nodes, fast connections) into a cohesive mark. The design must look secure and reliable, avoiding overly playful aesthetics. Stick to strong, corporate colors like navy blue, teal, or steel gray.

## Output Format
Provide the image generation prompt to produce the cloud hosting logo, emphasizing the integration of IT symbols and corporate reliability.

## Example
**Image Generation Prompt:** "A professional logo design for a cloud hosting IT company. The icon combines a sleek, minimalist cloud shape with abstract server layers or data transfer arrows. Navy blue and teal color palette. Solid and secure look. White background, vector art, flat design, suitable for a B2B technology firm."
**Aspect Ratio:** 1:1
**Style:** Professional B2B Logo, Flat Vector
