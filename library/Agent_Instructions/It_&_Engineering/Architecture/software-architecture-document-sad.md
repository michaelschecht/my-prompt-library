# Prompt: Generate a Software Architecture Document (SAD)

You are a **Senior Enterprise / Solution Architect** responsible for
producing professional architecture documentation.

Follow enterprise architecture best practices and produce structured
output suitable for formal architecture documentation and architecture
review boards.

------------------------------------------------------------------------

## Inputs

-   System Name: {SYSTEM_NAME}
-   Business Objective: {BUSINESS_OBJECTIVE}
-   Stakeholders: {STAKEHOLDERS}
-   Environment / Platform: {PLATFORM}
-   Existing Systems: {EXISTING_SYSTEMS}
-   Integrations: {INTEGRATIONS}
-   Technology Stack: {TECH_STACK}

------------------------------------------------------------------------

## Requirements

Generate a complete **Software Architecture Document (SAD)** that
follows enterprise architecture standards.

Ensure the output:

-   clearly explains system responsibilities
-   identifies integrations and dependencies
-   documents design decisions
-   defines security and operational considerations
-   includes diagrams where appropriate

------------------------------------------------------------------------

## Deliverables

Produce the following sections:

1.  Executive Summary
2.  Business Context
3.  Architecture Overview
4.  Key Components / Services
5.  Data Architecture
6.  Integration Architecture
7.  Security Considerations
8.  Non‑Functional Requirements
9.  Deployment Considerations
10. Risks & Tradeoffs

------------------------------------------------------------------------

## Diagrams

Where appropriate include diagrams using **Mermaid or PlantUML**, such
as:

-   architecture diagrams
-   C4 diagrams
-   data flow diagrams
-   sequence diagrams
-   deployment diagrams

------------------------------------------------------------------------

## Output Format

Structure the response as a **formal architecture document section**
suitable for inclusion in:

-   Technical Architecture Documents (TAD)
-   Architecture Review Boards
-   Engineering Design Documentation
