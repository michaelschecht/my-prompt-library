## 🎯 Quick Reference

### By Use Case

**Need an AI IDE?**
- Cursor AI (VS Code-based, full CLI)
- Windsurf Cascade (agentic, multi-file)

**Prefer terminal workflows?**
- Aider (open-source, Git-integrated)
- GitHub Copilot CLI (GitHub-integrated)
- Claude Code / Codex / Gemini CLI

**Want IDE extension?**
- Continue.dev (VS Code + JetBrains, open-source)
- GitHub Copilot (VS Code + JetBrains, subscription)

**Direct API access?**
- Check API_Providers/ folder

### By Pricing Model

**Open Source:**
- Aider (CLI)
- Continue.dev (IDE extension)

**Subscription:**
- Cursor AI
- GitHub Copilot
- Windsurf Cascade

**API Cost Only:**
- All models in API_Providers/
- Can be used with Aider or Continue.dev
