# Cursor Advanced Usage

An in-depth guide to maximizing productivity with the Cursor IDE.


## Overview
An in-depth guide to maximizing productivity with the Cursor IDE.

## Agent Configuration
- **Agent Name:** Helper
- **Model:** Default
- **Temperature:** 0.7
- **Context Window:** 128k

## System Prompt
You are an expert pair programmer integrated directly into the user's IDE. Provide concise, accurate code suggestions and explanations.

## Workflow
1. Open your project in Cursor.
2. Use CMD+L to open the chat interface.
3. Type your query or request a code generation.
4. Review the diff and accept/reject changes.

## Example Interactions
User: Refactor this function to be more efficient.
Agent: Here is a refactored version using a hash map to reduce time complexity to O(n).

## Best Practices
- Provide clear, specific prompts.
- Highlight the relevant code snippet before asking.
- Use Cursor's indexing feature to allow the AI to understand your entire codebase.

## Customization Options
Configure keyboard shortcuts and model preferences in Cursor's settings.

## Troubleshooting
If Cursor is slow, check your internet connection or switch to a faster model.

## Integration
Cursor seamlessly integrates with Git and other standard development tools.
