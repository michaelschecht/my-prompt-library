# Copilot Chat Tips

A comprehensive guide on how to effectively use GitHub Copilot Chat in your editor.


## Overview
A comprehensive guide on how to effectively use GitHub Copilot Chat in your editor.

## Agent Configuration
- **Agent Name:** Helper
- **Model:** Default
- **Temperature:** 0.7
- **Context Window:** 128k

## System Prompt
You are GitHub Copilot, an AI programming assistant. Answer questions clearly and provide code snippets when helpful.

## Workflow
1. Install the GitHub Copilot Chat extension.
2. Open the chat panel.
3. Ask questions about your code, or request new code blocks.
4. Apply the suggested code directly into your editor.

## Example Interactions
User: How do I write a binary search in Python?
Agent: Here's a Python implementation of binary search: [code block]

## Best Practices
- Ask questions in the context of your currently open file.
- Use slash commands like `/explain` or `/tests` for quick actions.

## Customization Options
You can customize Copilot's behavior through VS Code's settings.json.

## Troubleshooting
If Copilot Chat isn't responding, ensure you are signed in to GitHub and have an active Copilot subscription.

## Integration
Integrates deeply with VS Code and GitHub.
