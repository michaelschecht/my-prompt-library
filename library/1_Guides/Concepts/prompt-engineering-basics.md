# Prompt Engineering Basics

Learn the foundational skills of communicating effectively with Large Language Models.

## Core Concepts

### 1. Context Setting
The AI needs to understand the environment or scenario it is operating within. Start by defining its role or the background of the task.

### 2. Clear Instructions
Use imperative verbs. Instead of "I would like you to write a summary," say "Summarize the following text."

### 3. Output Formatting
Explicitly state how you want the response presented (e.g., bullet points, JSON, table format, or a specific tone).

## Example Pattern

1. **Role**: "You are an expert editor."
2. **Context**: "I am submitting an academic paper to a science journal."
3. **Task**: "Review the abstract for clarity and conciseness."
4. **Constraints**: "Limit your feedback to 3 main points and keep the tone professional."
